## ML2 Gym
Custom made, modified ewnvironments used by ML2 for reinforcement learning

### PyCon Walker
Modified version of bipedal walker from OpenAI gym. 

Two player playable. 
For more information, refer to the `README.md` in pycon_walker directory.
